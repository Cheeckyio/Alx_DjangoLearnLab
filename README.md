# LibraryProject

A Django-based library management system.

## Setup Instructions

### Prerequisites
- Python 3.x installed on your system
- pip (Python package installer)

### Installation

1. Install Django:
```bash
   pip install django
```

2. Navigate to the project directory:
```bash
   cd LibraryProject
```

3. Run the development server:
```bash
   python manage.py runserver
```

4. Open your browser and visit:
```
   http://127.0.0.1:8000/
```

## Project Structure

- `manage.py`: Command-line utility for interacting with the Django project
- `settings.py`: Configuration file for the Django project
- `urls.py`: URL declarations for the project (URL routing)

## Purpose

This project serves as the foundation for developing Django applications focused on library management functionality.
